## Notes

- `cheb` is very large (3435 `.m` files). The audit caps at 200 files per domain for runtime reasons.
- `dsp` and `opt` READMEs contain no GitHub URL. The repos were cloned during a previous session but the URL was not recorded. Mark as **unknown**; reconstruct from domain descriptions above if needed.
- Three bio-related corpora (`nw-bio`, `dna-bio`, `bioseq`) had 0 `.m` files at scan time — likely cloned as shallow repos with no MATLAB source at the root level; check subdirectories after re-cloning.
- `odeprobe`, `bio`, and `oop` are Mjolnir-internal and do not require an external fetch.
